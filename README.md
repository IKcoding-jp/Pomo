# ポモドーロタイマー

バニラJS（HTML / CSS / JavaScript）で作ったポモドーロタイマーです。  
JavaScript学習の一環として、自分が毎日使う道具を自作しています。

## 機能

- 25分カウントダウン
- スタートボタンで開始
- 分:秒 形式で表示（例：`24:09`）

## 今後追加予定

- タイマーが0になったら通知
- スタート / ストップ切り替え
- コーヒーをテーマにしたデザイン
- 残り時間を円グラフで表示するアニメーション

## 技術スタック

- HTML5
- CSS3
- JavaScript（バニラ・フレームワーク不使用）

## 動かし方

```
index.html をブラウザで開く
```
